# Tools

API 驱动的工具集，每个工具独立部署。

## 工具列表

| 工具 | 说明 |
|------|------|
| `rate-board` | 汇率看板 |
| `atlas-info` | 地图信息 |
| `radio-atlas` | 电台地图 |
| `sky-brief` | 天气简报 |
| `trending-atlas` | 趋势地图 |
| `repo-scope` | 仓库分析 |
| `node-atlas` | 节点地图 |
| `hour-bridge` | 时区换算 |
| `offday-planner` | 假期规划 |
| `art-gallery` | 艺术画廊 |
| `earthquake` | 地震监测 |
| `dev-tools` | 开发工具 |

## 技术栈

前端: Vue 3 + Vite  
后端: 各工具独立 API

## 部署

各工具独立构建部署，互不依赖。
