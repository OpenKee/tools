# Tools

开箱即用的小工具集合。

## 工具

- **Rate Board** — 实时汇率看板
- **Atlas Info** — 地点信息查询
- **Radio Atlas** — 全球电台地图
- **Sky Brief** — 天气简报
- **Trending Atlas** — 趋势热力图
- **Repo Scope** — GitHub 仓库分析
- **Node Atlas** — 节点分布地图
- **Hour Bridge** — 时区换算
- **Offday Planner** — 假期规划
- **Art Gallery** — 随机艺术画廊
- **Earthquake** — 实时地震监测
- **Dev Tools** — 开发者工具箱

每个工具独立运行，互不依赖。
