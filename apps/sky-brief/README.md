# Sky Brief

Sky Brief is a practical weather dashboard powered by free Open-Meteo APIs.

## Features

- place search via free geocoding API
- current temperature and conditions
- air quality summary
- wind and humidity
- 12-hour outlook
- 7-day forecast
- plain-English daily suggestion
- optional browser geolocation

## APIs used

- Open-Meteo Geocoding API
- Open-Meteo Forecast API
- Open-Meteo Air Quality API
